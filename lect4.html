Links / Hyperlinks

Links allow users to move from one page to another. They are one of the most important parts of web design because they connect different pages and resources together.

Absolute Link: This type of link directs users to another website.

Visit Google

Relative Link: This link connects to another page within the same website.

Go to another page

Jump Link: This allows navigation within the same page.

Images

Images help improve the visual appearance of a webpage. They can also provide additional meaning and context to the content. The alt attribute is important because it describes the image when it cannot be displayed.

Image as Link

An image can also act as a clickable link. When the user clicks the image, it redirects them to another page.
