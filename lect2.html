Lecture 2: HTML Structures

This section explains the basic structure and concepts behind HTML pages.

HTML is used to structure web content like text, images, and links.

A webpage usually starts with <!DOCTYPE html> and is divided into head and body sections.

Elements are the building blocks of a webpage, while tags define them.

Attributes give extra information to elements like alignment, color, or behavior.

Semantic tags like header, section, and footer help organize content clearly.

HTML entities are used for special characters like <, >, and &.
