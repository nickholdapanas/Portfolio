Introduction to CSS, selectors, syntax, and styling concepts.

Introduction to CSS3

CSS (Cascading Style Sheets) is used to control the appearance of web pages. It separates structure (HTML) from design (CSS) for better organization and maintainability.

CSS was developed by W3C to solve problems where HTML was being used for styling.

Why CSS Was Created

Before CSS, HTML contained formatting tags like font and color which caused messy and repetitive code. CSS fixed this by allowing styling rules to be written in one place.

CSS Syntax

A CSS rule consists of a selector and a declaration block. Each declaration contains a property and value.
